# 🖋️ Manglish Pro (Smart Malayalam Notes)

A fast, sleek, and privacy-focused Malayalam transliteration tool. Type in Manglish (English characters) and watch it instantly convert to Malayalam script. Perfect for creators, students, and anyone who wants to write in Malayalam without a complex keyboard layout.

![Status](https://img.shields.io/badge/Status-Live-success)

## ✨ Key Features

- **🚀 Instant Transliteration:** Powered by Google Input Tools API for high accuracy.
- **📁 Note Management (CRUD):** Create, Save, and Delete multiple notes. All data stays in your browser's `LocalStorage`.
- **⌨️ Keyboard Navigation:** Navigate through word suggestions using arrow keys and apply them with `Space`.
- **🌓 Modern UI & Dark Mode:** A clean, "Zen" writing environment with a toggle-able dark theme.
- **🌐 Bilingual Toggle:** Switch between English and Malayalam modes instantly with `Ctrl + Space`.
- **📤 Export Options:** Copy to clipboard or download your notes as `.txt` files.

## 🛠️ How to Use

1. **Type:** Just start typing in Manglish (e.g., *namskaram*).
2. **Convert:** Press `Space` or `Enter` to convert the last word to Malayalam.
3. **Select:** Use `Left/Right Arrows` to pick a different spelling from the suggestions bar.
4. **Manage:** Use the sidebar to switch between different notes or create a new one.
5. **Shortcuts:**
    - `Ctrl + Space`: Toggle between English and Malayalam mode.
    - `Esc`: Clear the suggestions bar.
